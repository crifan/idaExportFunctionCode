# 快速上手
 
本指南帮助你在5分钟内完成IDA函数代码导出插件的安装和首次运行。
 
## 前置条件
 
- **IDA Pro 7.0+**（需要Hex-Rays反编译器许可证才能导出`.c`伪代码）
- IDA自带Python环境（无需额外安装）
 
## 第一步：下载插件
 
从GitHub仓库下载以下两个文件：
- `idaExportFunctionCode.py` （主脚本）
- `config.json` （配置文件）
 
```
https://github.com/crifan/idaExportFunctionCode
```
 
## 第二步：放置文件
 
将下载的两个文件放到**同一个目录**下，建议放置位置：
- IDA的`scripts`目录，或
- 任意你方便找到的位置
 
> **注意**：`config.json` 必须与 `idaExportFunctionCode.py` 在同一目录
 
## 第三步：编辑配置
 
打开 `config.json`，配置要导出的函数：
 
```json
{
  "functionList": [
    { "startAddress": "0xA389" },
    { "funcName": "sub_B000" }
  ]
}
```
 
最简配置只需指定 `startAddress`（十六进制地址）或 `funcName`（函数名）之一。
 
## 第四步：在IDA中运行
 
1. 用IDA Pro打开目标二进制文件
2. 菜单栏：**File** → **Script file...**
3. 选择 `idaExportFunctionCode.py` 并运行
 
## 第五步：查看输出
 
导出文件位于二进制文件同目录下的 `exportedCode/function/` 文件夹：
 
```
exportedCode/function/
├── sub_A389.c      # 伪代码
├── sub_A389.asm    # 汇编代码
└── sub_A389.bin    # 原始字节
```
 
## 验证成功
 
IDA输出窗口显示类似信息即表示成功：
```
[idaExportFunctionCode] Exported: sub_A389.c
[idaExportFunctionCode] Exported: sub_A389.asm
[idaExportFunctionCode] Done. Total exported: 3 files
```
